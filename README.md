U.S. States Game

An interactive U.S. States knowledge game built with Python, Turtle Graphics, and Pandas.

The objective is to correctly guess all 50 U.S. states. Each correct guess is displayed in its corresponding location on the map. If you choose to exit before completing the game, a CSV file containing the states you missed is generated so you can practice them later.

Features
Interactive map using Turtle Graphics
Real-time display of correctly guessed states
Score tracker showing progress
Prevents duplicate scoring for repeated guesses
Generates a states_to_learn.csv file with missed states
Technologies Used
Python 3
Turtle
Pandas
CSV files
